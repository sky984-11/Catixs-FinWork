# 项目看板功能说明

项目看板用于跟踪客户项目从规划、进行、验收到完成的全过程。它把项目负责人、客户、进度、ETA、子任务、讨论、文件和飞书提醒集中到一个页面，适合日常项目推进和早会同步。

## 入口

前端菜单进入：`项目看板`

飞书每日总结中的项目标题和子任务标题支持点击跳转：

- 点击项目标题：打开项目详情。
- 点击子任务标题：打开项目详情并定位到对应子任务。

## 看板视图

看板按项目状态分为四列：

- 规划中：还在准备资源、排期或等待启动。
- 进行中：正在实施。
- 验收中：等待客户、内部或交付结果确认。
- 已完成：项目已经结束。

项目卡片上会显示客户、负责人、优先级、健康度、ETA、进度和未完成子任务。鼠标悬浮到项目卡片时，会展示未完成子任务的提示列表，方便快速判断下一步要处理什么。

## 新增项目

点击右上角 `新增项目` 创建项目。

必填字段：

- 项目名称
- 客户
- 负责人

建议填写：

- 项目编号：用于和合同、订单或内部交付编号对应。
- 合同编号：便于财务和合同追踪。
- 截止日期：用于看板逾期判断和每日总结。
- 项目说明：记录交付范围、特殊要求和关键背景。

客户下拉框支持中文、英文和拼音模糊搜索，并优先展示客户全称。

## 项目进度

打开项目详情后，可以在概览中拖拽项目进度条调整进度。

进度颜色会随完成度渐变：

- 低进度偏红色，表示刚开始或风险较高。
- 中间进度逐步过渡到橙色、黄色。
- 接近完成时变为绿色。

拖拽后系统会自动保存，无需额外点击保存。

## 子任务

每个项目可以创建多个子任务。子任务用于拆分实际执行动作，例如：

- 确认资源
- 等待供应商回复
- 收发光信息
- Patching
- 客户验收

子任务支持：

- 设置负责人
- 设置 ETA 时间
- 编辑标题、负责人、ETA、备注
- 勾选完成
- 删除
- 回复讨论
- 上传或添加相关文件链接

子任务 ETA 会用于飞书提醒。

## 文件和链接

项目文件支持两种方式：

- 上传本地文件
- 添加外部链接

外部链接适合飞书文档、飞书文件夹、共享盘目录等无法直接上传的资料。链接必须以 `http://` 或 `https://` 开头。

子任务也支持上传文件和添加链接，适合把某个交付动作相关的资料挂在对应任务下面。

## 共享项目

默认情况下：

- 普通用户只能看到自己负责的项目。
- `admin` 可以看到所有项目。

项目卡片上的共享图标可以把项目共享给一个或多个用户。被共享用户可以在自己的项目看板中看到该项目。

适用场景：

- 项目负责人需要让协同人员看到项目。
- 交付、NOC、财务等角色需要共同跟进同一个项目。
- 临时交接项目。

## 飞书提醒

系统支持两类飞书通知。

### 子任务 ETA 提醒

未完成子任务会按 ETA 触发提醒：

- 距离 ETA 还有一天：发送临近到期提醒。
- 到达 ETA：发送到期提醒。

如果项目已经完成或归档，该项目下的子任务不会继续提醒。

如果修改了子任务 ETA，系统会重置该子任务的提醒状态，按新的 ETA 重新判断。

### 每日项目总结

每天早上 8:30，系统会把规划中、进行中、验收中的项目发送到飞书。

当前每日总结只统计和 `Azure` 有关的项目，包括：

- 项目负责人为 Azure。
- 子任务负责人包含 Azure。
- 项目共享用户包含 Azure。

每日总结包含：

- 负责人汇总
- 项目数
- 未完成子任务数
- 按状态分组的项目列表
- 项目编号
- 子任务编号
- 项目 ETA
- 子任务 ETA
- 可点击跳转的项目和子任务链接

飞书链接固定使用：

```text
https://finwork.catixs.net
```

## 推荐使用方式

1. 创建项目时先选客户、负责人和状态。
2. 把交付动作拆成子任务，并给每个子任务设置负责人和 ETA。
3. 每天通过看板查看逾期和风险项目。
4. 项目推进时更新进度条和子任务状态。
5. 需要多人协同时，用共享功能把项目共享给对应用户。
6. 项目结束后把状态改为已完成，避免继续产生子任务提醒。

## 常见问题

### 为什么我看不到某个项目？

普通用户只能看到自己负责或被共享的项目。需要项目负责人把项目共享给你，或者由 `admin` 查看。

### 为什么子任务没有飞书提醒？

通常有几种原因：

- 子任务已经完成。
- 项目已经完成或归档。
- 子任务没有设置 ETA。
- 飞书 webhook 未配置或发送失败。
- 该子任务已经发送过对应阶段提醒。

### 为什么每日总结里没有某个项目？

每日总结只包含规划中、进行中、验收中的项目，并且当前只统计和 `Azure` 有关的项目。已完成或归档项目不会进入每日总结。

### 为什么飞书链接打不开？

确认后端配置中的 `WEB_BASE_URL` 为：

```text
https://finwork.catixs.net
```

修改配置后需要重启后端服务。
